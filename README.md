# XAI-платформа для анализа рекомендательных систем

## О проекте

Этот проект представляет собой концепцию системы, которая позволяет объяснять рекомендации (например, "почему пользователю показали этот товар") и анализировать их качество.

Цель — сделать рекомендательные системы более прозрачными для аналитиков, разработчиков и бизнеса.

---

## Какие задачи решает система

- объяснение рекомендаций в понятной форме  
- анализ факторов, влияющих на рекомендации  
- выявление некорректных или подозрительных рекомендаций  
- сравнение разных моделей  
- формирование отчетов для анализа  

---

## Моя роль в проекте

- проработал бизнес-логику системы  
- описал пользовательские сценарии (use cases, user stories)  
- сформировал функциональные и нефункциональные требования  
- спроектировал архитектуру системы (C4 model)  
- разработал модель данных (ERD)  
- описал API взаимодействие

---

## Почему это важно

Рекомендательные системы часто работают как "чёрный ящик".  
Это создаёт проблемы:

- сложно понять, почему принимаются решения  
- трудно находить ошибки  
- снижается доверие пользователей  

Этот проект показывает, как можно сделать такие системы более прозрачными и управляемыми.
